#  Create an App

1. Create and clone repository.
2. Add .gitignore (see below).
3. Copy files from snap-app-template.
4. Rename project.
4.1. Rename top item in project navigator.
4.2. Change bundle identifier.


## .gitignore

```
# macOS
.DS_Store
.netrc

# Xcode
xcuserdata/
contents.xcworkspacedata
/.build
```
